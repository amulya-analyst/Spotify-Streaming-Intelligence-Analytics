# Spotify Streaming Intelligence Analytics

## Author
Amulya

## Project Overview
End-to-end Spotify streaming intelligence analysis covering audio 
features, song popularity trends and subscription analytics using 
Python, SQL and Power BI to generate actionable music industry insights.

## Tools Used
- Python (pandas, matplotlib, seaborn)
- SQL (MySQL)
- Power BI
- Excel

## Key Insights
- Identified top performing genres by streams and revenue
- Analyzed audio features (danceability, energy, tempo) impact on popularity
- Compared Free vs Premium subscription revenue performance
- Tracked streaming trends across multiple countries

## Key Analysis
- Audio features analysis (danceability, energy, loudness, tempo, valence)
- Song popularity categorization (Low, Average, Popular)
- Subscription plan performance and revenue analysis
- Country-wise streaming and revenue trends
- Artist and genre performance comparison

## Dataset
- 8 genres, multiple artists
- Countries: Japan, Canada, Brazil and more
- Subscription plans: Free and Premium
- Features: 19 columns including audio metrics and business data
